Thoughts & Learned: This problem was pretty hard, but I learned that when you encounter a 2D array matrix problem, the first step is to traverse the matrix.


class Solution {
  public void gameOfLife(int[][] board) {
    int m = board.length;
    int n = board[0].length;

    // Print initial board
    for (int i = 0; i < m; i++) {
      for (int j = 0; j < n; j++) {
        System.out.print(board[i][j] + " ");
      }
      System.out.println();
    }
  }
}
